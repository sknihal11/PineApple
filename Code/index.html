Smart Campus Navigation Platform

Campus Navigation System

A smart navigation platform that helps students and visitors find classrooms, departments, labs, and important campus facilities through outdoor route guidance and indoor room-level directions.

Start Navigation Open Admin Panel

Outdoor Navigation

Guides users to the correct academic block and facility using campus map routing.

Indoor Guidance

Provides floor-wise, corridor-wise, and classroom-level instructions after arrival.

Nearby Discovery

Uses approximate live location to recommend nearby facilities and key campus places.

Admin Updates

Allows room allocation and navigation details to be updated dynamically.

Project Capabilities

This system reduces confusion caused by shifting departments, changing room allocations, and unclear timetable-based location information.

Search by Classroom, Department, Year, Section Students can search destinations in multiple ways instead of relying only on room numbers.
Landmark-Assisted Identification Helps infer probable location and nearby facilities through landmark-based selection.
Approximate GPS Support Uses live location for nearby suggestions and maps it to known campus anchor points when needed.
Dynamic Classroom Mapping Admins can add, update, and correct room navigation information whenever allocations change.
Suggested demo flow: open navigation → search classroom → show outdoor route → display indoor guidance → show nearby places → show admin update.
